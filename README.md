## About / 关于

I build things from the compiler up: a self-hosting language, an AI agent runtime, and small reliable tools. I care about **understanding every line I ship** — Agents align syntax and logic, but the code is mine.

我从编译器往上造东西：一门自举语言、一个 AI agent 运行时、若干可靠小工具。我在意**理解自己交付的每一行** —— Agent 对齐语法与逻辑，但代码是我亲手写的。

Like a good containment protocol, my repositories follow one consistent structure — documented once, applied everywhere.

就像一套稳妥的规程，我的仓库遵循同一套结构 —— 定义一次，处处适用。
